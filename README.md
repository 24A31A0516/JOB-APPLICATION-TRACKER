Job Application Tracker and Resume Skill Analyzer

Overview

I developed this project as a front-end web application to help students and job seekers manage their job applications, analyze their resumes, and prepare for interviews in one place.

The application allows users to track job applications, identify relevant job roles based on skills, and practice interview questions.

---

Features

Authentication

- User signup and login system
- Data stored using LocalStorage
- Separate data for each user

Job Application Tracker

- Add job details such as company, role, type, round, status, and date
- View all applications in a table format
- Delete job entries
- Displays statistics like total applications, interviews, and rejections

Resume Skill Analyzer

- Upload resume files (PDF, DOC, DOCX, TXT)
- Detects technical skills from the resume (basic simulation)
- Suggests suitable job roles based on detected skills
- Option to manually enter skills

Interview Preparation

Company Questions

- Shows interview questions for companies like Amazon, TCS, Google, Microsoft, and Infosys

Practice Questions

- Total 120 questions divided into:
  - Technical
  - Aptitude
  - Verbal
  - Non-Tech
- Multiple choice format with instant feedback
- Shows correct answers
- Tracks progress for each section

Progress Tracking

- Displays progress using progress bars
- Tracks number of correct answers
- Provides feedback based on performance
